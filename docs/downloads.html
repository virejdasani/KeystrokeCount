Keystroke Count

Data for your keystrokes

View on GitHub Download for MacOS

Keystroke Count keeps all your keystroke data locally stored and NO data including your keystrokes is ever collected.

Installation on MacOS

1. Download Keystroke Count For MacOS from above

2. Once downloaded, double-click the file

3. This will extract the app. Simply drag the app into the Applications folder on your machine.

4. If you are prompted with something like “Keystroke Count” cannot be opened because the developer cannot be verified., click cancel.

5. Go to system preferences -> Security and Privacy -> General -> and click Open Anyway.
